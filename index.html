Everything you need

Built for real software.
Not enterprise bloat.

Every feature NexAuth ships is something an indie dev actually needs. No fluff, no sales calls, no enterprise pricing.

🔑
License Keys & Tiers
Free, premium, and admin tiers out of the box. Redeem codes, trial periods, and instant upgrades — all manageable from your admin panel.
💻
HWID Locking
Bind licenses to hardware fingerprints automatically. Users can self-serve HWID resets after a cooldown, or you reset them instantly from the admin panel.
⏱️
Trial Management
Create time-limited trials with automatic expiry. Set durations in minutes, hours, days, or weeks. Extend or revoke from the admin panel anytime.
🛡️
Ban & Suspend
Temporary suspensions with reasons and auto-expiry, or permanent bans. Users see your message on login. Lift restrictions with one click.
📍
IP Tracking
See the last 3 login IPs for every user. Admin accounts are always redacted. Useful for spotting account sharing or suspicious activity.
🔐
Password Resets
Admin-gated reset codes with 48-hour expiry. Users request resets, you get a notification, issue the code, done. No email server required.
🎛️
Embedded Admin Panel
Your admin panel lives inside your app behind a tier check. No separate login, no web dashboard to maintain, no extra hosting costs.
📶
Offline Grace Period
Users stay authenticated for up to 7 days without an internet connection. Perfect for apps used in low-connectivity environments.
⚡
JWT Sessions
Industry-standard JWT authentication with configurable expiry. Tokens are signed server-side — clients can never forge or modify them.
Simple by design

Up in 5 minutes.

No DevOps degree required. If you can run a Python script, you can ship NexAuth.

1
Deploy Your Backend
Click deploy on Render. Your auth server is live in under 2 minutes with a Postgres database included.
2
Add to Your App
Drop auth.py and login_window.py into your project. Configure your API URL. That's the entire integration.
3
Gate Your Features
Use auth.is_premium() and auth.current_tier to gate any feature in your app. One line of code per check.
4
Ship It
Compile to EXE, distribute however you want. Your users register, pay, and get access — automatically.
